# WaveFrag-Acoustic-Camera
WaveFrag 声学相机提供高性能声学成像与信号处理系统，适用于实验室和个人开发者进行多通道声源定位与分析。

## Features
- 多通道麦克风阵列采集
- 实时声源定位与声压图生成
- 支持 Python / MATLAB SDK 接口
- 数据可视化功能
- 支持 FFT、Beamforming 等信号处理算法
- 灵活配置采样率和处理参数
